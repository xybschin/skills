# Skills

A flat collection of agent skills.

## Skills

| Skill | Purpose |
|-------|---------|
| **cavecrew** | Delegate to caveman-style subagents to save context |
| **caveman** | Ultra-compressed communication mode (~75% fewer tokens) |
| **caveman-commit** | Ultra-compressed commit message generator |
| **caveman-compress** | Compress memory files into caveman format |
| **caveman-help** | Quick-reference card for all caveman modes |
| **caveman-review** | Ultra-compressed code review comments |
| **caveman-stats** | Show real token usage and savings for the session |
| **commit** | Generate conventional commit messages and create commits |
| **grill-me** | Interview relentlessly about a plan until shared understanding |
| **grill-with-docs** | Grill a plan against the existing domain model and update docs |
| **improve-codebase-architecture** | Find architecture improvement opportunities in a codebase |
| **setup-skills** | Set up the agent skills block and issue tracker configuration |
| **to-issues** | Break a plan into independently-grabbable issues |
| **to-prd** | Turn conversation context into a PRD |
| **triage** | Triage issues through a state machine |
| **write-a-skill** | Create new agent skills |

## Getting Started

**Note**: Several skills require `setup-skills` to be run first in your repository.
